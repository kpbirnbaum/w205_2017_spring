1. Go into the directory exercise_2/extweetwordcount/src/spouts
2. Open vi editor for spouts and enter twitter credentials
3. Move back up into the extweetwordcount directory
4. type ‘sparse run’ and hit enter
5. hit control + C to stop the stream
6. type ‘python finalresults.py specifiedword’ to get the count of specified word
7. type ‘python histogram.py rangestartinteger,rangeendinteger’ in order to get all words within specified range.
